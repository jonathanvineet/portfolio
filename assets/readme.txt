ASSETS FOLDER - PLACEHOLDER FILES
==================================

This folder contains placeholder images and icons for the portfolio website.

TO CUSTOMIZE:
-------------

1. IMAGES:
   - Replace photo.jpg with your actual portrait photo (400x400px recommended)
   - Replace iot.jpg with a background image for IoT projects section (1920x1080px)
   - Replace software.jpg with a background image for Software projects section (1920x1080px)

2. ICONS:
   All SVG icons in the /icons folder can be customized or replaced:
   - favicon.svg - Browser tab icon
   - mail.svg - Email icon for contact section
   - linkedin.svg - LinkedIn icon
   - github.svg - GitHub icon
   - close.svg - Modal close button icon

3. CURRENT FILES:
   All current image files are SVG placeholders with themed colors.
   They will display properly but should be replaced with real photos for production.

4. SKILLS SECTION:
   Skill animations are intentionally left minimal for user injection via JavaScript.
   Use the VineetPortfolio.animateSkill() function in script.js to add custom animations.

RECOMMENDED IMAGE FORMATS:
--------------------------
- Portrait: JPG or PNG, 400x400px minimum
- Background images: JPG, 1920x1080px minimum
- Icons: SVG (for scalability) or PNG with transparency

NOTES:
------
- Keep file names the same when replacing to avoid updating HTML references
- Optimize images for web (compress before uploading)
- Maintain aspect ratios for best display
