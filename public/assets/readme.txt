ASSETS FOLDER - PLACEHOLDER FILES
==================================

This folder contains placeholder images and icons for the portfolio website.

TO CUSTOMIZE:
-------------

1. IMAGES:
   - Replace photo.svg with your actual portrait image
   - Replace iot.svg with a background image for IoT projects section
   - Replace software.svg with a background image for Software projects section

2. ICONS:
   All SVG icons in the /icons folder can be customized or replaced:
   - favicon.svg - Browser tab icon
   - mail.svg - Email icon for contact section
   - linkedin.svg - LinkedIn icon
   - github.svg - GitHub icon
   - close.svg - Modal close button icon

RECOMMENDED IMAGE FORMATS:
- Portrait: JPG or PNG, 400x400px minimum
- Background images: JPG, 1920x1080px minimum
- Icons: SVG (for scalability) or PNG with transparency

NOTES:
- Keep file names the same when replacing to avoid updating HTML references
